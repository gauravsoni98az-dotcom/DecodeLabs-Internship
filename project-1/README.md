# Project 1 - Responsive Frontend Interface 
This is my first project from the DecodeLabs Full Stack Development Internship. I built a responsive web interface using just HTML, CSS, and JavaScript — no frameworks, no shortcuts.

## About
The idea was simple — learn how to build a proper web page from scratch. I focused on making it look good on all screen sizes and writing clean, readable code.

## What I Built
- A fully responsive webpage that works on mobile, tablet, and desktop
- Used CSS Grid and Flexbox for layout
- Wrote vanilla JavaScript for interactivity
- Kept the design clean and simple

## Tech Used
- HTML5
- CSS3
- JavaScript

## Files
- `index.html` — main page
- `style.css` — all the styling
- `script.js` — javascript logic
- `images/` — images used in the project

## What I Learned
This project taught me how responsive design actually works under the hood. Writing media queries manually and figuring out layouts without a framework gave me a solid foundation before jumping into backend development.
