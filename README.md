# Project 1 - Responsive Frontend Interface 🎨

A responsive frontend web interface built as part of the **DecodeLabs Full Stack Development Internship Program **.

## 📌 About the Project

This project focuses on building a clean, responsive, and user-friendly web interface using pure HTML, CSS, and JavaScript — no frameworks. The goal was to master the fundamentals of frontend development and responsive design before moving to backend technologies.

## 🚀 Features

- Fully responsive layout that works on mobile, tablet, and desktop
- Mobile-first design approach
- CSS Grid and Flexbox for layout management
- Semantic HTML5 structure
- Clean and modern UI following 2025 design aesthetics
- Basic JavaScript for interactivity and state management

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and semantic landmarks |
| CSS3 | Styling, Grid, Flexbox, Media Queries |
| JavaScript | Interactivity and basic state management |

## 📁 Project Structure

```
project-1/
├── index.html      # Main HTML file
├── style.css       # Stylesheet
├── script.js       # JavaScript logic
└── images/         # Image assets
```

## 🎯 Key Concepts Applied

- Semantic HTML5 landmarks (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`)
- Responsive breakpoints: Mobile → Tablet (768px) → Desktop (1024px)
- CSS Grid for macro layouts
- Flexbox for micro components
- Mobile-first strategy using `min-width` media queries
